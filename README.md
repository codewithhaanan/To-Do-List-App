# Todo App

This is a practice Todo application built using **HTML, CSS, and JavaScript**.  
It helped me learn and practice the following concepts:

- DOM manipulation (adding/removing tasks dynamically)
- Event handling (click events for buttons and tasks)
- LocalStorage (saving tasks even after page reload)
- Responsive layout with CSS

## Features

- Add new tasks
- Delete tasks
- Persist tasks using localStorage
- Simple and user-friendly UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## How to Use

1. Clone the repository
2. Open `index.html` in a browser
3. Start adding tasks
